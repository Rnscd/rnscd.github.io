Projetos Desenvolvi diversos aplicativos em projetos pessoais, academicos e para clientes como Freelancer. Aqui estão alguns demonstrando diversos tipos de skills que desenvolvi ao longo do tempo.

Sparkle - Dating & Social

Aplicativo de rede social e relacionamento desenvolvido em Kotlin, com foco em interações baseadas em geolocalização. Os usuários podem filtrar perfis por localização, publicar textos e fotos, comentar, criar e compartilhar perfis, além de enviar mensagens privadas, entre outras funcionalidades sociais. O app foi construído com a arquitetura MVVM, utilizando Jetpack Compose para a interface do usuário. Toda a infraestrutura online é baseada no Firebase. Também foram utilizadas bibliotecas como Dagger Hilt (injeção de dependência), DataStore (armazenamento local), entre outras ferramentas modernas do ecossistema Android.

Iris Livros App

Este é um aplicativo Android feito com Compose que apresenta informações de livros, como titulo, autor, página etc, com base na pesquisa do usuário usando a Google Books API. Desenvolvi na arquitetura MVVM e utilizei as bibliotecas Retrofit para interagir com a API, JUnit para testes autonomos, ViewModel com stateflow para gerenciar o estado do App, Coil para carregar imagens da API e Gson para converter JSON, entre outras.

São Paulo Recomenda App

Android App feito em Kotlin e Compose que mostra recomendações dos melhores locais da cidade de Sao Paulo. Foi desenvolvido utilizando as melhores práticas modernas recomendadas pela google, como: arquitetura MVVM; utliza ViewModel e StateFlow pra atualizar dados que são apresentados. Testes Autonomos; Junit para testes Locais e instrumentais, Reutilização de Composes; reutiliza de composes menores como de páginas inteiras, Material Theming, etc.

Hotel Santo Antonio App

Este projeto foi desenvolvido em um time de 4 pessoas na faculdade com metodologia ágil. O aplicativo foi desenvolvido em Java para Android, e faz conexão com um banco de dados MS SQL Server. É um app para um hotél e tem funções de criação e autenticação de conta, busca e reserva de quartos do hotél, e gerenciamento das resrevas feitas.
